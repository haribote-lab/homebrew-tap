# homebrew-tap

Homebrew formulae for [haribote-lab](https://github.com/haribote-lab) projects.

## Install

```bash
brew install haribote-lab/tap/gha
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [gha](https://github.com/haribote-lab/github-app-cli) | Proxy `gh` CLI commands with GitHub App authentication |

## How It Works

Formulae are automatically updated via GitHub Actions when new releases are published in source repositories. Updates are submitted as pull requests for review.
